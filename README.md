\# Лабораторна робота №1

\## Проектування системи "Квітковий магазин"



\## Постановка задачі

Метою роботи є розробка UML-моделі інформаційної системи "Квітковий магазин".

Система повинна забезпечувати облік товарів, клієнтів та замовлень.

Необхідно визначити основні сутності, їх атрибути та зв’язки між ними.
